HTML-документ — это обычный текстовый документ, созданный в любом текстовом редакторе, например в блокноте, и оформленный в соответсвии с правилами языка HTML

Для файлов, содержащих HTML-документы, используется расширение HTML ИЛИ HTM

Началось все для HTML в конце 1980-х годов, когда у ученных из Европейской лаборатории элементарных частиц возникла необходимость обмениваться множеством различных докумнетов при помощи быстро развивающейся сети Интернет. Тогда приходилось придумьта такой способ публикации данных в сети, чтобы к любому нужному документу можно было легко получить доступ и чтобы документы отображались на всех компьютерах одинаковым образом.
